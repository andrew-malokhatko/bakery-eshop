# bakery-eshop

To run the application install VS code Live server extension by Ritwick Dey

then right click on index.html and select Open with live server (alternatively press Alt+L+O)

This should open the website in your browser

If the website was not opened automatically, you can find it at http://127.0.0.1:5500/

To stop the Live Server press ctrl+shift+p and select stop life server
